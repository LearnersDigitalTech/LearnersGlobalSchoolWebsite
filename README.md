# Learners Global School Website

This is the official website for Learners Global School, built with Next.js (App Router), TypeScript, and SCSS Modules.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components and feature-specific components.
- `src/styles`: Global styles, variables, and mixins.
- `src/data`: Seed JSON data for CMS integration.

## Features

- **Responsive Design**: Mobile-first approach using SCSS mixins.
- **Accessibility**: WCAG AA compliant, including skip links, ARIA labels, and keyboard navigation.
- **Performance**: Optimized images and static generation for pages.
- **Theming**: CSS variables for easy theme customization.

## Pages

- **Home**: `/`
- **About**: `/about`
- **Courses & Admissions**: `/courses-admissions`
- **Life at LGS**: `/life`
- **Connect**: `/connect`

## Deployment

To build the application for production:

```bash
npm run build
```

To start the production server:

```bash
npm start
```
